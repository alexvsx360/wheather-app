# מערכת מזג אוויר - שיעור 8

יישום מזג אוויר פשוט המבוסס על JavaScript למתחילים.

## מטרת הפרויקט

פרויקט זה הוא יישום מעשי של הנושאים שנלמדו בשיעור JavaScript למתחילים. הפרויקט כולל:
- שימוש ב-API חיצוני (Open-Meteo) לקבלת נתוני מזג אוויר
- עבודה עם Axios לבקשות HTTP
- שימוש ב-Bootstrap לעיצוב UI
- ניהול מצב (state management) בסיסי
- עבודה עם אירועים (events) וטופסים
- שימוש ב-Vite ככלי בנייה
- ארגון קוד מודולרי עם ES6 Modules

## טכנולוגיות

- **Vite** - כלי בנייה ופיתוח מהיר
- **Bootstrap 5** - ספריית CSS לעיצוב
- **Axios** - ספרייה לבקשות HTTP
- **Open-Meteo API** - API חינמי לנתוני מזג אוויר

## התקנה והרצה

1. התקן את התלויות:
```bash
npm install
```

2. הרץ את השרת לפיתוח:
```bash
npm run dev
```

3. בנה את הפרויקט לייצור:
```bash
npm run build
```

4. תצוגה מקדימה של הבנייה:
```bash
npm run preview
```

## ארכיטקטורה מודולרית

הפרויקט מאורגן במבנה מודולרי המאפשר:
- **הפרדת אחריות** - כל מודול אחראי על תחום ספציפי
- **קוד רב-פעמי** - פונקציות שניתן להשתמש בהן במקומות שונים
- **תחזוקה קלה** - קל למצוא ולתקן באגים
- **בדיקות** - כל מודול ניתן לבדיקה בנפרד

### תיאור המודולים:

- **`api/weatherApi.js`** - מכיל את כל הקריאות ל-API (גיאוקודינג ומזג אוויר)
- **`utils/domUtils.js`** - מחזיר אובייקט עם כל הפניות לאלמנטי DOM
- **`utils/formatters.js`** - פונקציות לעיצוב נתונים (תאריכים, מספרים וכו')
- **`ui/messageHandler.js`** - ניהול הודעות והתראות למשתמש
- **`ui/resultDisplay.js`** - תצוגה ועדכון של תוצאות מזג האוויר
- **`ui/themeHandler.js`** - ניהול מעבר בין מצב לילה/יום
